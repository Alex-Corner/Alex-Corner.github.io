Exercises 1: Modular Arithmetic

Exercises

Calculators aren’t needed for these questions - you can always use them to check, but you should focus on getting to grips with modular arithmetic as a way of thinking before reaching for the technology. Try to avoid doing arithmetic with numbers bigger than 100 - see if you can reduce them first.

  1. Use modular arithmetic to do the following calculations - you can always check your answer on a calendar after. There is a table in the notes which you may find useful.

    1. What day of the week will 31st December fall on this year?

    2. What day of the week will 14th February fall on next year?

    1. Calculate \(100\) modulo \(24\).

    2. Calculate \(1000\) modulo \(24\).

    3. What time of day will it be \(3000\) hours from now?

  3. Find the value of \(7^{137} \ (\mathrm{mod}\ 11)\).

  4. Find the value of \(7^{137} \ (\mathrm{mod}\ 8)\). (Hint: Don’t just jump into this. If you think about this in the right way, it ends up being very easy.)

  5. Find the last \(2\) digits of \(3^{124}\).

Solutions

    1. A Tuesday. Work out the number of days left until 31st December. Then reduce this number modulo \(7\).

    2. A Wednesday. Do the same as above but for 14th February.

    1. \(100 = 4\ (\mathrm{mod}\ 24)\)

    2. Calculate \(1000\) modulo \(24\) by multiplying the previous calculation by \(10\) and reducing it again modulo \(24\).

    3. This can be calculated from the previous step by multiplying the result by \(3\) and reducing it again.

  3. First break up the power into powers of \(2\): \(7^{137} = 7^{128} \times 7^8 \times 7^1\). Then build it up in stages:

    Then

    \[ 7^{137} = 7^{128} \times 7^8 \times 7^1 = 9 \times 9 \times 7 = 81 \times 7 = 4 \times 7 = 28 = 6 \ (\mathrm{mod}\ 11). \]

  4. If you look at this the right way, it’s very easy. Remember that we can use negative integers as well as positive integers. Since we are working modulo \(8\), then we have that

    \[ 7 = -1 \ (\mathrm{mod}\ 8). \]

    From this we can then conclude that

    \[ 7^{137} = (-1)^{137} = -1 = 7 \ (\mathrm{mod}\ 8). \]

  5. To find this, work out \(3^{124} \ (\mathrm{mod}\ 100)\) in a similar way to before. The last two digits are \(8\) and \(1\).
